Font-family

Font-family(с англ. семейство шрифта) - В значении этого свойства указывается список названий шрифтов, перечисленных через запятую. В начале списка располагают самый желаемый шрифт, затем менее желаемый, а в самом конце списка — общий тип шрифта. Font-style

Font-style(с англ. стиль шрифта) - это CSS-свойство определяющее каким образом шрифт должен быть стилизирован, будь то это normal, italic или oblique.

Мы рассмотрим значения свойств:

  1. ! italick
  2. ! normal
  3. ! oblique

Значение normal является значением по умолчанию, т.е. ничего не меняет. italick делает курсивный текст. oblique - наклонный текст. В отличие от italic, который имитирует рукописный текст, наклонный представляет собой обычный текст, буквы которого наклонены в сторону.

Text-decoration

Это свойство, с помощью которого можно задать дополнительное оформление текста.

  1. underline — подчёркивание;
  2. line-through — зачёркивание;
  3. overline — надчёркивание;
  4. none — убирает вышеперечисленные эффекты.

Text-transform

С помощью Text-transform можно также управлять регистром символов: делать буквы строчными (маленькими) или заглавными (большими)

  1. lowercase — все строчные;
  2. uppercase — все заглавные;
  3. capitalize — каждое слово начинается с большой буквы;
  4. none — отменяет изменение регистра.

p {
text-transform: uppercase;
}

Box-shadow

Позволяет добавлять элементу одну или несколько теней, указывая их параметры через запятую. Свойство box-shadow позволяет вам задать тень для почти любого элемента. Если элементу задано свойство border-radius, то тень также будет с закруглёнными углами.

Оформление и перобразование нашего текста для более приятного чтения

line-height, letter-spacing, text-indent, word-spacing, font-size - данные свойства предназначены для того, чтобы разрядить текст и пользователю было приятно его читать

  1. line-height — высота линии строки, создаёт расстояние между строками, что значительно облегчает чтение;
  2. letter-spacing — расстояние между буквами, неплохо разряжает текст;
  3. text-indent — отступ в начале, а если более просто - создаёт красную строку. Эстетика;
  4. word-spacing —расстояние между словами. Неплохое дополнение для letter-spacing;
  5. font-size — задаёт размер шрифту.

Overflow

Определяет, необходимо ли для переполненного блочного элемента содержимое обрезать, предоставить полосы прокрутки или просто отобразить. Имеет следующие значения:

  1. visible — По умолчанию. Содержимое не обрезается, может отображаться снаружи блока, в котором оно расположено;
  2. hidden — Контент обрезается, без предоставления прокрутки;
  3. scroll — Содержимое обрезается и браузер использует элементы прокрутки, не важно было ли обрезано содержимое или нет. Это избегает многих проблем касательно появления прокруток и их исчезновения в динамичной среде. Принтеры могут печатать переполненное содержимое;
  4. auto — Зависит от агента пользователя. Браузеры такие как Firefox предоставляют прокрутку, если содержимое переполняет блок.

overline: auto; в блоке в сочетании с overflow: hidden; во всей оболочке(wreaper) создаёт вертикальный скролл без горизонтального.

Внешние и внутренние отступы

margin является внешним отступом, т.е. расстояние от одного объекта до другого объекта; padding является внутренним отступом, т.е. расстояние от границ объекта до его содержимого.

Внешние отступы:

  4. padding-left: — значение

Высота и ширина

height и width существуют для того чтобы объекту можно было задать определенную ширину и высоту.С их помощью вы можете устанавливать фиксированные размеры элементов.

Тема довольна обширная и её проще разобрать на практике. Видео по разбору этих свойств и других находится здесь.

Сразу стоит запомнить, что у свойств width и height есть особенность — они не включают в себя значения margin, padding и border. Значение, которое вы установите для width/height, будет означать лишь ширину/высоту области содержимого элемента.

Свойство height может быть удобным, если надо точно контролировать, например, высоту изображения. Однако, если в контейнере будет содержаться текст или любой другой контент, у которого может варьироваться высота, крайне не рекомендуется устанавливать фиксированную высоту для контейнера, так как подобная верстка может привести к неожиданному результату — контент будет отображаться поверх другого содержимого.

Вместо фиксированной высоты использовать height: auto — эта запись означает, что высота объекта будет рассчитываться автоматически, в зависимости от содержимого, которое он содержит.

Другой способ избежать развала верстки — использовать запись overflow: auto. В таком случае, если высота содержимого будет превышать значение height своего контейнера, браузер добавит к контейнеру полосу прокрутки.

Ширину элемента удобно указывать для дальнейшего выравнивания блока.

Выравнивание при помощи margin

Указав ширину блочному объекту, его можно выравнивать по горизонтали, используя margin.

  1. margin: 0 auto — выравнивание по центру;
  2. margin: 0 0 0 auto — выравнивание по правому краю;
  3. margin: auto 0 0 0 — выравнивание по левому краю.

Display

Определяет тип отображения элемента. Крайне многогранное свойство. Может применяться для вырывания блока из страницы, для превращения из строчного в блочный элемент и наоборот, для образования гибридного блока и т.д. Мы рассмотрим несколько важнейших свойств, которые проходили на уроке.

  1. inline — делает тег строчным;
  2. block: 0 0 0 auto — делает тег блочным;
  3. inline-block — делает тег строчным, но даёт возможность применять к нему блочные свойства;
  4. none — вырывает блок из страницы, не резервируя его;

Border

- это граница объета. Имеет как универсальную запись, так и несколько записей, каждая из которых направлена на стилизацию одной стороны:

  1. border-top — 1px solid(прямая линия) red;
  2. border-left — 1px dotted(точки) red;
  3. border-bottom — 1px dashed(пунктир) red;
  4. border-right — 1px double(двойная линия) red.

Универсальная запись указывает одним свойством значение для всез сторон: border: 10px solid red;

Border-radius

Свойство, округляющее границы объекта. Указывается как в %, так и в px. Закруглённость каждого угла определяется с помощью одного или двух радиусов, определяя его форму: круг или эллипс.

Opacity

Устанавливает непрозрачность элемента. Непрозрачность - это степень, в которой содержимое скрывается за элементом, является противоположностью прозрачности.

opacity: 0; - полностью скрывает объект, при этом с ним можно взаимодействовать.

visibility

Cкрывает или показывает элемент без изменения разметки документа. Но в отличии от opacity: 0; с объектом нельзя взаимодействовать. Значения:

  1. visible — по умолчанию. Объект видно;
  2. hidden — объект исчезает, резервируя место. При этом возможность взаимодействия отсутствует.

Outline

Работает точно так же, как и border, только находится за границей блока. border резервирует место внутри объекта в зависимости от ширины границы, а outline обводит тег.

outline не влияет на размеры и положение самого элемента, не занимает место, не влияет на окружающие элементы и отображается поверх них.

Нельзя задать параметры линии на отдельных сторонах элемента, outline применяется сразу ко всем четырём сторонам.

{
outline: 1px solid red;
}

text-align

Описывает, как выравниваются объекты внутри блока по горизонтали. Я описал как работают значнения с текстом.

  1. left — выравнивание по левому краю блока, это значение по умолчанию;
  2. right — по правому краю блока;
  3. center — по центру блока;
  4. justify — по ширине блока, при этом слова в строке будут размещаться так, чтобы занять равномерно всё пространство строки (пробелы между словами в таком случае становятся неравномерными, так как браузер «растягивает» слова в строке).
